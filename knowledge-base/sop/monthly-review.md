# Monthly Review SOP

The process for conducting the monthly performance review across all active clients and campaigns. The monthly review is deeper than the weekly report. It covers funnel health, lead quality, offer economics, and strategy adjustments.

---

## Ownership

- Primary owner: Analytics Agent (compiles and analyzes)
- Support: CFO Agent (financials), Marketing Director (strategy decisions), Paid Ads Agent (campaign adjustments)

---

## When to Run

On the last business day of each month, or the first business day of the new month. Cover the full prior calendar month.

---

## Step 1: Pull Data

**Owner:** Analytics Agent

Data to pull for each client:
- Meta Ads: All metrics by campaign, ad set, and top/bottom ads for the month
- Google Ads (if active): Same breakdown
- CRM/GHL: Leads by source, pipeline stage counts, Booked/Showed/Closed rates and totals
- Revenue: Closed revenue for the month, compared to goal
- Email (if active): Full month open and click rates, unsubscribes

---

---

## Step 3: Run the Funnel Analysis

**Owner:** Analytics Agent

Run skills/analytics-optimization/funnel-analysis.md.

Identify:
- Where is the biggest drop-off in the funnel?
- What is the revenue gap from underperforming stages?
- Which stage gets fixed first?

---

## Step 4: Run the Lead Quality Review

**Owner:** Analytics Agent

Run skills/analytics-optimization/lead-quality-review.md.

Identify:
- Are leads converting downstream at target rates?
- Which source produces the best quality leads?
- Is there an audience-copy mismatch?

---

## Step 5: Run the Ad Performance Review

**Owner:** Analytics Agent + Paid Ads Agent

Run skills/analytics-optimization/ad-performance-review.md with full month data.

Identify:
- Which campaigns, ad sets, and ads performed best?
- What creative types are working?
- What is the creative rotation schedule for next month?

---

## Step 6: Run the Content Performance Review (if applicable)

**Owner:** Analytics Agent

Run skills/analytics-optimization/content-performance-review.md.

Identify:
- Which content pillars performed best?
- Which formats generated the most engagement?
- What should the content calendar focus on next month?

---

## Step 7: Compile the Monthly Report

**Owner:** Analytics Agent

Combine findings from steps 2-6 into a single monthly report. Structure:

1. Executive summary (3-5 bullets: what happened this month in plain language)
2. KPI scorecard (the 10-metric table with vs. target and vs. prior month)
3. Top 3 wins this month
4. Top 3 problems this month
5. Funnel health summary
6. Lead quality summary
7. Recommended actions for next month

---

## Step 8: Next Actions

**Owner:** Analytics Agent + Marketing Director

Run skills/analytics-optimization/next-actions.md.

Output: Prioritized action list for next month, with owner and due date per item.

---

## Step 9: Deliver to Client (if applicable)

**Owner:** Marketing Director

Review the report before sending. The QA Agent runs a formatting-review.md on the final document before delivery.

Schedule a monthly review call with the client to walk through findings.

---

## Status

Active SOP. Run on the last business day of each month.
