## How To Add Visual Rules

1. Add hex codes for primary and secondary colors.
2. Add the primary and secondary font names.
3. Add 2 to 3 sentences describing the visual style and mood.
4. Note any hard rules (e.g., "never use the logo on a dark background").
5. Add reference images or examples to `knowledge-base/examples/`.
