# Knowledge Base Map

Where to find things in the knowledge base and what goes in each folder.

---

## Folder Guide

### brand/

What belongs here:
- `voice-rules.md` (already exists) - Global writing rules for all agents
- `visual-rules.md` (placeholder) - Brand colors, fonts, design standards
- Additional brand guidelines as they are created

Agents that read this: All agents, especially Copywriter and QA Agent.

---

### offers/

What belongs here:
- One file per offer or product (e.g., `flagship-course.md`, `ads-management-service.md`)
- Each file should include: name, transformation promise, price, stack, guarantee, urgency
- Client-specific offers should note the client name

Agents that read this: Offer Agent, Copywriter, Paid Ads Agent, CFO Agent.

---

### clients/

What belongs here:
- One subfolder per client (e.g., `clients/acme-inc/`)
- Each folder should have a README with: business description, niche, audience, current campaigns, goals

Agents that read this: All agents when doing client-specific work.

---

### frameworks/

What belongs here:
- Strategy frameworks and models the team uses
- Examples: Value Ladder framework, Campaign launch checklist framework, Conversion optimization model
- Keep each framework in its own file

Agents that read this: Marketing Director, Offer Agent, Funnel Agent, Copywriter.

---

### examples/

What belongs here:
- Example copy and creative for reference
- Examples should be organized by type: `ads/`, `landing-pages/`, `emails/`, `social/`
- Include context: what it is, what niche, why it works

Agents that read this: Copywriter, Creative Design Agent, Social Media Agent.

---

### swipe-files/

What belongs here:
- Competitor ads (as text descriptions)
- Strong headlines and hooks collected from research
- Email subject lines worth referencing
- Landing page structures that convert
- Do not copy copyrighted material verbatim

Agents that read this: Copywriter, Market Research Agent, Paid Ads Agent.

---

### case-studies/

What belongs here:
- Client wins with before/after metrics
- Campaign results that can be referenced in copy
- Process case studies showing how the team solved a problem

Agents that read this: Copywriter, Marketing Director, Offer Agent.

---

### sop/

What belongs here:
- Standard operating procedures for recurring agency tasks
- Onboarding, campaign launch, reporting, offboarding
- Keep each SOP in its own file with numbered steps

Agents that read this: n8n Automation Agent, CRM Follow-Up Agent, Marketing Director.

---

### incoming-docs/

What belongs here:
- Raw files that need to be reviewed and filed
- Anything uploaded by the client or team that has not been categorized yet

Agents that read this: None until files are reviewed and moved.

---

## Uploaded Source Files

Raw uploads live in `uploads/extracted/` not in `knowledge-base/`.

See `docs/imported-files-audit.md` for a map of what is in uploads/extracted/ and where it should go.
