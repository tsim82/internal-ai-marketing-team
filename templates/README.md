# Templates

Reusable template files for common agency deliverables. Use these as starting points - fill in the brackets and check the boxes before sending anything to a client, agent, or production.

---

## Intake Templates

Used to collect information at the start of a client relationship or project.

| File | When to Use |
|------|-------------|
| `intake/client-intake.md` | Send to a new client after signing. Collects all business, offer, audience, and technical context before the kickoff call. |
| `intake/project-intake.md` | Use internally when starting a new campaign or deliverable for an existing client. |

---

## Brief Templates

Used to give agents everything they need before starting a build task.

| File | When to Use |
|------|-------------|
| `briefs/campaign-brief.md` | Before building a paid ad campaign. The Marketing Director and Paid Ads Agent use this. |
| `briefs/copy-brief.md` | Before the Copywriter starts any written deliverable. |
| `briefs/creative-brief.md` | Before the Creative Design Agent or Video Agent starts any visual deliverable. |

---

## Report Templates

Used to structure weekly and monthly performance deliverables.

| File | When to Use |
|------|-------------|
| `reports/weekly-report.md` | Every week. Completed by the Analytics Agent. |
| `reports/monthly-report.md` | End of each month. Completed by the Analytics Agent and reviewed by the Marketing Director before client delivery. |

---

## Checklist Templates

Used before launching or handing off any deliverable.

| File | When to Use |
|------|-------------|
| `checklists/pre-launch-checklist.md` | Before any campaign, funnel, or automation goes live. Completed by the QA Agent. |
| `checklists/handoff-checklist.md` | Before delivering any output to the next agent or to a client. |

---

## Prompt Templates

Ready-to-use Claude Code prompts for the most common workflows. Copy, fill in the brackets, and paste into Claude Code.

| File | What It Does |
|------|-------------|
| `prompts/new-client-kickoff.md` | Kicks off the full onboarding process for a new client. Routes to Marketing Director. |
| `prompts/campaign-launch.md` | Starts a new paid campaign build. Routes to all relevant agents. |
| `prompts/weekly-review.md` | Runs the weekly performance review. Routes to Analytics Agent. |
| `prompts/research-request.md` | Starts a market research session. Routes to Market Research Agent. |

---

## How to Use These Templates

1. Open the template file
2. Fill in all bracketed fields `[like this]`
3. Check all applicable checkboxes
4. Remove or mark N/A any sections that do not apply
5. Do not send anything to a client with unfilled brackets
