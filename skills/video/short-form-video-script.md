# Short-Form Video Script

## Purpose

Write a complete, execution-ready script for a 15-90 second short-form video (TikTok, Instagram Reel, YouTube Short, Facebook Reel). The script includes the spoken word, visual direction, and text overlay notes so the creator or video editor can produce the video without additional briefing.

## Used By

- Video Agent (primary)
- Copywriter (writes the spoken script)
- Social Media Agent (briefs the topic)

## Trigger

Use when a short-form video slot is on the content calendar or when a long-form video needs to be adapted into short-form clips.

## Inputs Needed

- Topic and key point the video makes - required
- Platform and target video length - required
- Audience profile - required
- Content pillar (education, social proof, brand, engagement, offer) - required
- Any hook angle from hook research - helpful
- Whether this is an original video or a clip from a longer piece - required

## Process

**Step 1: Select the hook**

The first 1-3 seconds determine whether anyone watches the rest. The hook must be immediate.

Hook types ranked by short-form effectiveness:

| Hook type | Example | Best for |
|-----------|---------|---------|
| Text on screen + strong opener | "The real reason your ads aren't working" | Cold audiences, education |
| Bold claim spoken | "I made $40k from one Instagram post. Here's exactly how." | Social proof, authority |
| Question | "Why does everyone say X, but almost nobody does Y?" | Curiosity, engagement |
| Contrarian | "Stop [common advice]. It's actually hurting you." | Pattern interrupt |
| Proof opener | "This client went from 2 leads a week to 20 in 30 days." | Social proof |
| Visual hook | An action, a transformation, or an unexpected visual happening immediately | When visual is the hook |

**Step 2: Plan the structure**

Short-form structure for a 30-60 second video:

| Timing | Section | What happens |
|--------|---------|-------------|
| 0-3s | HOOK | Spoken hook + optional text on screen |
| 3-8s | PATTERN INTERRUPT | Something that keeps them watching (reveal, setup, tension) |
| 8-45s | PAYOFF | The actual content: the teaching, the story, the insight |
| 45-60s | CTA | Tell them what to do next (follow, comment, DM, link) |

For 15-second videos: Hook (0-3s) + Payoff (3-12s) + CTA (12-15s). No pattern interrupt section.

**Step 3: Write the script**

Write word for word. Do not write "talk about your experience with X." Write the actual words.

Short-form scripts should:
- Use short sentences (8-12 words max)
- Sound like how the person actually talks
- Pause between ideas (write [PAUSE] where needed)
- Match the platform energy (TikTok is higher energy than LinkedIn)

**Step 4: Add visual direction and text overlays**

For each scripted line or section, note:
- [TEXT ON SCREEN: "text here"] - what appears as a caption or overlay
- [VISUAL: what the camera should show or what B-roll to use]
- [CUT: if this is a cut point in the edit]

Text overlays are often more important than the spoken word on short-form because many users watch without sound.

## Output Format

```
SHORT-FORM VIDEO SCRIPT - [Client] - [Platform] - [Date]

TOPIC: [What the video is about]
CONTENT PILLAR: [Education / Social proof / Brand / Engagement / Offer]
TARGET LENGTH: [15s / 30s / 45s / 60s / 90s]
PLATFORM: [TikTok / Instagram Reel / YouTube Short / Facebook Reel]
HOOK TYPE: [From hook types table above]

---

SCRIPT:

[0:00-0:03] HOOK
SPOKEN: "[Hook line - word for word]"
TEXT ON SCREEN: "[Text overlay if any]"
VISUAL: [What to show - close-up face / action / graphic]

[0:03-0:08] PATTERN INTERRUPT (if 30s+)
SPOKEN: "[Next line]"
TEXT ON SCREEN: "[If applicable]"
VISUAL: [Direction]

[0:08-0:45] PAYOFF
SPOKEN: "[Main content - word for word, line by line]"
TEXT ON SCREEN: "[Key points as they are spoken]"
VISUAL: [Direction for each beat]

[0:45-0:60] CTA
SPOKEN: "[CTA line]"
TEXT ON SCREEN: "[CTA text overlay]"
VISUAL: [Direction]

---

TOTAL WORD COUNT: [X]
ESTIMATED TIME AT NORMAL PACE: [X seconds]

CAPTION: See short-form-caption.md

NOTES FOR CREATOR:
- Energy level: [High / Medium / Calm]
- Any specific gestures, props, or visual elements needed
- If repurposed from a longer video: timestamp [X:XX to X:XX]
```

## Quality Check

- [ ] Hook is in the first 1-3 seconds, not a greeting or intro
- [ ] Script is written word for word (no "ad lib here")
- [ ] Text overlay directions are included for key lines
- [ ] CTA is at the end and is specific
- [ ] Estimated time is within the target length
- [ ] Short sentences only (8-12 words max)
- [ ] Visual direction is specific, not "show something relevant"

## Status

Phase 3 complete.
