# Content Calendar

## Purpose

Build a monthly content calendar that maps out every social post across all platforms for a 4-week period. The calendar balances content pillars, platform formats, and posting cadence. The output is a structured table the Social Media Agent uses to brief content and the Copywriter uses to write each post.

## Used By

- Social Media Agent (primary)
- Copywriter (writes the post copy from each row in the calendar)
- Marketing Director (reviews before the month starts)

## Trigger

Use at the start of each month or content planning cycle, or when the team is posting without a structured plan.

## Inputs Needed

- Content pillars for this client (from the Social Media Agent's strategy) - required
- Active platforms (Instagram, Facebook, LinkedIn, short-form, or combination) - required
- Posting cadence (how many times per week per platform) - required
- Offer or campaign focus for the month (if any) - required
- Any existing high-performing content to repurpose - helpful
- Key dates (launches, promotions, deadlines) in the month - helpful

## Process

**Step 1: Confirm the pillar split**

Standard pillar split:
- Education: 40%
- Social proof: 20%
- Brand/story: 20%
- Engagement: 10%
- Offer/promotion: 10%

Adjust if the month has a launch or promotion. A launch month might shift to 25% offer content for weeks 3-4.

**Step 2: Map posts to a weekly rhythm**

Assign pillar types to specific days so the week has variety. Example weekly rhythm for 5x/week Instagram:

| Day | Pillar |
|----|-------|
| Monday | Education |
| Tuesday | Social proof |
| Wednesday | Education |
| Thursday | Brand/story |
| Friday | Engagement or offer |

Adjust for each platform and cadence.

**Step 3: Assign topics to each slot**

For each post slot, assign a specific topic. Not "tips post" - a real topic the Copywriter can write from.

Good: "3 reasons most home service businesses lose leads in the first hour"
Bad: "lead generation tips"

Topics should pull from:
- Objection research (address common hesitations)
- Hook research (use emotional triggers)
- Recent questions or conversations from clients
- Upcoming offers or campaigns

**Step 4: Flag any platform-specific formats**

Note if a post slot should be a Reel, a carousel, a static image, or text-only. This tells the Creative Design Agent what assets to prep.

**Step 5: Mark key promotional dates**

If there is a launch, a deadline, or a promotion this month, mark which posts tie to it and ensure the pillar mix reflects the campaign.

## Output Format

```
CONTENT CALENDAR - [Client] - [Month/Year]

ACTIVE PLATFORMS: [Instagram / Facebook / LinkedIn / TikTok / YouTube Shorts]
POSTING CADENCE: [X posts/week per platform]
MONTHLY FOCUS / CAMPAIGN: [If any]

PILLAR SPLIT THIS MONTH:
  Education: [X]%
  Social proof: [X]%
  Brand/story: [X]%
  Engagement: [X]%
  Offer/promotion: [X]%

---

WEEK 1 ([Date range]):

| Day | Platform | Pillar | Topic | Format | Status |
|-----|---------|--------|-------|--------|--------|
| Mon [date] | Instagram | Education | [Specific topic] | Static / Reel / Carousel | [Draft / Copy needed / Ready] |
| Tue [date] | Facebook | Social proof | [Specific topic / client result] | Static | [...] |
| Wed [date] | LinkedIn | Education | [Specific topic] | Text + image | [...] |
| Thu [date] | Instagram | Brand/story | [Specific topic] | Reel | [...] |
| Fri [date] | Instagram + Facebook | Engagement | [Specific question or topic] | Static | [...] |

WEEK 2 ([Date range]):
[Same format]

WEEK 3 ([Date range]):
[Same format]

WEEK 4 ([Date range]):
[Same format]

---

PROMOTIONAL DATES THIS MONTH:
[Date]: [Event or deadline - flag which posts tie to this]

ASSET NEEDS FOR CREATIVE DESIGN AGENT:
- [X] static images needed
- [X] Reel concepts needed
- [X] Carousels needed
- Brief these by [Date] to allow production time

COPY BRIEF FOR COPYWRITER:
Posts needing copy by [Date]: [List post slots]
```

## Quality Check

- [ ] Every pillar is represented across the month
- [ ] Each topic is specific enough for the Copywriter to write without follow-up
- [ ] Format is specified per post (static, Reel, carousel, text)
- [ ] Promotional dates are flagged and reflected in the pillar mix
- [ ] Asset needs are communicated to the Creative Design Agent before copy writing begins
- [ ] Copy deadlines are set so production has time to turn around before publish dates
