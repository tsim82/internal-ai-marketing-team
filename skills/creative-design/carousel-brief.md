# Carousel Brief

## Purpose

Write the brief for a carousel post (multi-slide social content) on Instagram, LinkedIn, or Facebook. The brief defines every slide: what text appears, what visual concept goes with it, and what the swipe progression achieves. A carousel brief is a complete production document, not a topic suggestion.

## Used By

- Creative Design Agent (builds the slides from this brief)
- Copywriter (writes the slide copy within the brief or confirms the copy direction)
- Social Media Agent (briefs the topic and reviews)

## Trigger

Use when a content calendar slot calls for a carousel, when a framework or list needs to be presented in a visual format, or when educational content is better delivered across multiple slides than in a single post.

## Inputs Needed

- Content pillar and topic - required
- Number of slides (standard: 5-10 slides) - required
- Platform (Instagram, LinkedIn, Facebook) - required
- Brand visual rules - required
- Copy for each slide (or the direction to brief the Copywriter) - required
- Goal of the carousel (educate, generate saves, drive DMs, build brand) - required

## Process

**Step 1: Plan the carousel arc**

A carousel tells a story or delivers a framework across slides. The arc:

1. **Slide 1 (Hook slide)**: This is the cover. It must generate a swipe. One strong headline, no more. Often shows a result, a problem, or a bold claim.
2. **Slides 2-N (Content slides)**: One idea per slide. A slide with two ideas requires a reader to choose what to read first.
3. **Final slide (CTA slide)**: One clear action. DM, follow, save, link in bio, or comment.

**Step 2: Determine slide count**

Ideal carousel length by content type:
- Framework or process: one slide per step (3-7 slides + cover + CTA)
- List post: one slide per item (5-9 items + cover + CTA)
- Before/after: 3 slides (before, the shift, after/results)
- Story: 5-8 slides depending on arc complexity

Do not add slides to pad. Every slide must earn its place.

**Step 3: Write each slide brief**

For every slide:
- Slide number
- Slide type (Hook / Content / CTA)
- Headline or main text (the main thing the reader sees first)
- Supporting text (if any - subtext or brief explanation)
- Visual concept (what is behind or alongside the text)
- Color or style direction

**Step 4: Define the swipe hook**

Between slides, what creates the desire to keep swiping? Options:
- Incomplete thought ("Here's what most businesses get wrong about X...")
- Numbered list framing ("7 reasons. Here's reason 1.")
- Visual progression (a visual that completes across slides)
- Series structure ("Part 2 of 3...")

**Step 5: Design direction for the series**

A carousel should look cohesive across all slides:
- Consistent layout template per slide type
- Brand colors maintained
- Font hierarchy consistent (headline same size throughout, supporting text same size)
- One visual element that threads through all slides (a color stripe, icon style, or pattern)

## Output Format

```
CAROUSEL BRIEF - [Client] - [Topic] - [Date]

PLATFORM: [Instagram / LinkedIn / Facebook]
GOAL: [Saves / Education / DMs / Brand]
SLIDE COUNT: [N slides total]
CONTENT PILLAR: [Education / Social proof / Brand / Engagement / Offer]

---

CAROUSEL ARC:
[One sentence: what journey does the reader take from slide 1 to the last slide?]

---

SLIDE BRIEFS:

SLIDE 1 (HOOK):
  Headline: "[The one-line hook that makes someone swipe]"
  Supporting text: [If any / None]
  Visual: [Bold color background / Image / Pattern]
  Color: [Primary brand color or contrast color for maximum scroll-stop]
  Design note: [Keep it minimal - headline should dominate]

SLIDE 2 (CONTENT):
  Headline: "[One idea - short and scannable]"
  Supporting text: "[1-2 sentences max expanding on the headline]"
  Visual: [Icon / Illustration / Color block / Image]
  Color: [...]

SLIDE 3 (CONTENT):
  [Same format]

[Continue for each slide]

SLIDE [N] (CTA):
  CTA text: "[One clear action]"
  Supporting text: "[Optional: brief reason to take the action]"
  Visual: [Brand-consistent close]
  Color: [...]
  
---

VISUAL STYLE DIRECTION:

Background: [Clean white / Brand primary / Dark with contrast text]
Font: [Brand font - heading and body]
Icon style: [Line icons / Filled icons / None]
Illustration style: [Flat / Minimal / N/A]
Accent color: [Hex code for highlights or borders]
Layout: [Full bleed / Text-centered / Text-left with visual right]

---

COPY CONFIRMATION:
Copy written by: [Copywriter / Creative Design Agent writing their own copy]
If Copywriter: brief is attached / copy is embedded above

---

CAPTION (see social-media/instagram-post.md or linkedin-post.md for full caption):
Caption hook direction: "[The first line of the caption that accompanies this carousel]"
```

## Quality Check

- [ ] Slide 1 is a hook that makes someone want to swipe (not an intro or summary)
- [ ] One idea per content slide (no slide trying to do two things)
- [ ] Final slide has a single CTA
- [ ] Visual style is consistent across all slides
- [ ] Slide count is appropriate (not padded)
- [ ] Brief is specific enough to produce all slides without follow-up
