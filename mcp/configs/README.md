# MCP Configs

Contains example configuration files for setting up MCP server connections in Claude Code.

## Files

- `mcp.json.example` - Example Claude Code MCP config with all planned tools listed as placeholders.

## How To Use

1. Copy `mcp.json.example` to your Claude Code MCP config location.
2. Fill in the real values for the tools you want to connect.
3. Restart Claude Code to load the new connections.
4. Update `mcp/mcp-registry.md` to reflect which tools are now connected.

## Claude Code MCP Config Location

For project-level MCP config: `.claude/mcp.json` in the project root.
For user-level MCP config: `~/.claude/mcp.json`.

Check your Claude Code settings for the current config path.
