1. Drop zip files or raw docs into this folder.
2. Run the import process: extract zips, inspect contents, file relevant docs into `knowledge-base/`.
3. Update `docs/imported-files-audit.md` with a summary.
4. Do not edit files in `extracted/` - they are source files.
