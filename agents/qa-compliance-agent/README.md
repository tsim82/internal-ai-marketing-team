# QA / Compliance Agent

## What This Agent Does

Reviews every output before it goes live. Claims review, formatting check, brand voice check, broken link verification, and final handoff approval.

## When To Use It

- Before any copy, ad, or content goes to a client or goes live
- Before launching any funnel or campaign
- When checking a batch of social posts
- For end-of-sprint quality audits
