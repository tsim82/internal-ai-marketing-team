# Marketing Director Agent

## What This Agent Does

Plans projects, routes tasks to specialized agents, coordinates campaign execution, and reviews final outputs. This is the team's main coordinator and quality control layer.

## When To Use It

- Starting any new project or campaign
- When a task spans multiple agents
- When the scope is unclear and needs to be broken down first
- For final review before anything goes to a client
