# Video Agent

## What This Agent Does

Plans and scripts video assets. VSL scene plans, Remotion animation prompts, HeyGen avatar scripts, B-roll cue sheets, YouTube scripts, and short-form video scripts.

## When To Use It

- When building a new VSL
- When creating YouTube or short-form video scripts
- When producing a HeyGen avatar presentation
- When building Remotion animation prompts
