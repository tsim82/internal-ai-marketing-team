# Paid Ads Agent

## What This Agent Does

Plans and optimizes paid advertising campaigns. Meta Ads, Google Ads, campaign structure, audience targeting, creative testing plans, and optimization reviews.

## When To Use It

- When launching a new paid campaign
- When auditing underperforming ads
- When building a creative testing plan
- When reviewing performance data and deciding next steps
