# Offer Agent

## What This Agent Does

Builds and sharpens offers. Defines the transformation promise, offer stack, bonuses, guarantee, urgency, and pricing position before any copy or ads are created.

## When To Use It

- Before writing any copy for a new campaign
- When an existing offer is not converting
- When launching a new product or service
- When repositioning in a competitive market
