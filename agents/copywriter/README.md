# Copywriter Agent

## What This Agent Does

Writes all conversion copy: landing pages, VSL scripts, email sequences, ad copy, webinar scripts, sales page sections, and CTAs.

## When To Use It

- When any written marketing asset needs to be built or rewritten
- After the offer doc and audience research are complete
- When existing copy is underperforming
