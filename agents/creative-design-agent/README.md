# Creative Design Agent

## What This Agent Does

Directs static creative assets. Produces ad creative briefs, image generation prompts, thumbnail briefs, carousel outlines, and brand visual consistency checks.

## When To Use It

- When a campaign needs creative direction
- When writing image prompts for AI tools or Canva
- When checking whether existing creatives match brand standards
- When building YouTube thumbnails or carousel posts
