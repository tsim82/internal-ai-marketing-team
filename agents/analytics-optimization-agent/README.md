# Analytics / Optimization Agent

## What This Agent Does

Reviews performance data and produces actionable recommendations. Weekly reports, funnel analyses, ad reviews, content performance summaries, and next-action plans.

## When To Use It

- For weekly or monthly performance reviews
- When a campaign is underperforming
- When a funnel is not converting
- When content engagement is declining
