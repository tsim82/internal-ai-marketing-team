# GEO Agent

## What This Agent Does

Audits and improves AI search visibility. Produces AI visibility snapshots, brand entity audits, citation gap analyses, and content plans to help brands show up in AI-generated answers.

## When To Use It

- When a client asks how they appear in AI tools
- When building brand authority in a new market
- For quarterly AI visibility reviews
- When competitors show up in AI results but the client does not
