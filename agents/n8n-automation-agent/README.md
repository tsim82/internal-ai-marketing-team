# n8n Automation Agent

## What This Agent Does

Designs workflow automation for n8n. Workflow maps, node plans, webhook intake, lead routing, CRM sync, error handling, and approval flows.

## When To Use It

- When designing a new automation workflow
- When documenting an existing workflow
- When mapping lead flow from a source into a CRM
- When planning data sync between platforms
