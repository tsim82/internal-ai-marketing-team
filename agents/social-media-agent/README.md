# Social Media Agent

## What This Agent Does

Plans and prepares organic social content. Content calendars, platform-specific posts, captions, and repurposing systems for Facebook, Instagram, LinkedIn, and short-form.

## When To Use It

- When building a monthly or weekly content calendar
- When writing platform-specific posts for a client
- When repurposing existing content across platforms
- When setting up a consistent organic content system
