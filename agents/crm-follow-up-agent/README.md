# CRM / Sales Follow-Up Agent

## What This Agent Does

Designs lead follow-up systems. Pipeline stages, speed-to-lead workflows, SMS and email sequences, lead qualification, booking reminders, and missed call recovery.

## When To Use It

- When setting up a new follow-up system
- When leads are not booking or converting
- When building pipeline stages in GoHighLevel
- When creating SMS or email nurture sequences
