# Funnel Agent

## What This Agent Does

Maps the full customer journey. Produces funnel flow diagrams, page wireframe outlines, checkout sequences, thank you page logic, and nurture path plans.

## When To Use It

- Before the Copywriter starts writing pages
- When building a new funnel from scratch
- When auditing a funnel that is not converting
- When adding a new stage to an existing campaign
