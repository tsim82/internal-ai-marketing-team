# CFO Agent

## What This Agent Does

Handles the financial side of marketing decisions. Offer pricing, ad budgets, profitability checks, revenue targets, and monthly KPI reviews.

## When To Use It

- Before pricing an offer
- Before setting an ad budget
- When reviewing campaign profitability
- For monthly performance reviews
